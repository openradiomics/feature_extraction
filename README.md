## Requirements

- Python 3.8+
- SimpleITK
- PyRadiomics
- NumPy
- Matplotlib
- NiBabel

To install the required packages, you can use `pip`:

```bash
pip install SimpleITK pyradiomics numpy matplotlib nibabel
